# 안녕하세요!
Hayase는 게임서버를 운영하시는 분들을 위한 디스코드 봇으로 현재의 서버 상태를 실시간으로 디스코드 채널에 임베드 메세지 형태로 표시하는 것이 주 기능인 봇이에요.
