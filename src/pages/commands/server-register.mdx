# 게임 서버 등록
```/서버등록```

## 설명
실시간 현황을 표시할 게임 서버를 등록해요.
* 현황 채널이 등록되어 있어야 해요.
* 등록하는 시점에 게임 서버가 온라인이어야 해요.

## 명령어
예시: ```/서버등록 Minecraft 2b2t.org```

### 인자
* 게임 (필수)
  * 등록할 서버의 게임 코드
* 접속 주소 (필수)
  * 등록할 서버의 연결 가능한 주소
* 접속 포트 (선택)
  * 등록할 서버의 연결 가능한 포트
  * 가끔 연결 포트와 쿼리 포트가 불일치 하는 경우가 있어요, 그럴 경우 해당 게임 데디케이티드 서버의 문서를 참고하시거나 [공식 디스코드 서버](https://discord.gg/MwU25zdRFG)로 문의 주세요.
