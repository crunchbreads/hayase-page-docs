# 채널 설정 (자동)
```/채널설정-자동```

## 설명
봇의 초기 설정을 자동으로 해주는 명령어에요. 만약 n번째로 실행하신다면 기존 설정을 덮어쓰니 주의해주세요.

이 명령어가 자동으로 수행하는 작업은 다음과 같아요:
* 서버 현황 채널 생성 & 설정
* 서버 현황 채널에 메세지 전송

## 주의
이 명령어를 실행하면 기존의 서버 현황이 모두 초기화되어 새로 등록해야 하니 조심하세요.

## 명령어
예시: ```/채널설정-자동```
