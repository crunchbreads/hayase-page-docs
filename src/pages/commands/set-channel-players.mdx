# 채널 설정 (플레이어 카운트)
```/채널설정-플레이어```

## 주의
```
이 명령어로 활성화 되는 기능은 플레이어 수를 설정된 음성 채널 이름을 갱신하여 알려 주는 기능이에요.
디스코드 서버 감사 로그를 더럽히는 문제가 있어서 자동 채널설정으로는 활성화되지 않아요. 사용에 참고해주세요.
```

## 설명
현재 등록된 게임 서버들의 총 플레이어 수를 표시하는 채널을 설정해요.
* 등록할 채널은 음성 채널이어야 해요.
* 채널 이름을 변경해서 표시해요.
* **디스코드 서버 감사 로그에 지속적으로 기록이 남으니 유의하세요.**
* 본 기능을 비활성화 하고 싶으시면 [/설정](/commands/setting) 명령어로 비활성화 해주세요.

## 명령어
예시: ```/채널설정-플레이어 #플레이어```

### 인자
* 채널 (필수)
  * 플레이어 수를 표시할 채널이에요, 음성 채널이어야 해요.

### 권한
* [ ] 일반 유저
* [x] 관리자
