# 빠른 시작
## 1. 봇 초대하기
봇을 사용하기 위해 먼저 서버로 초대해야 해요.
1. [초대 링크](https://hayasebot.app/invite) 클릭
2. 초대하려는 서버를 선택하고 ```계속하기``` 버튼 클릭

![invite-1](/assets/static/invite-1.webp)

3. 권한을 확인하시고 ```승인``` 버튼 클릭

![invite-2](/assets/static/invite-2.webp)

![invite-3](/assets/static/invite-3.webp)

4. 커뮤니티 서버 설정이 되어 있지 않은 경우에 기본 언어가 영어에요, [/언어설정](/commands/set-language) 명령어로 한국어로 바꿔주세요.

![set-language](/assets/static/set-language.webp)

## 2. 채널 설정
### 자동
* **새로운 채널을 생성해요.**
* 가장 추천하는 방법이에요.

[/채널설정-자동](/commands/initialize) 명령어를 실행하면 모든게 자동으로 세팅돼요.

![set-channel-auto](/assets/static/set-channel-auto.webp)

### 수동
* **기존의 채널을 사용해요.**
* 채널 설정을 수동으로 해야 하는 경우에 사용해요.

```
- 주의: 채널 권한 설정
Hayase 봇이 서버 현황이 표시될 채널에 "메세지 보내기" 권한이 있어야 해요.
```

1. [/채널설정-서버현황](/commands/set-channel-status) 명령어로 서버 현황 메세지가 표시될 채널을 지정해요.

![set-channel-status](/assets/static/set-channel-status.webp)

2. [/채널설정-플레이어](/commands/set-channel-players) 명령어로 플레이어 수가 표시될 채널을 지정해요. 지정할 채널은 꼭 음성 채널이어야 해요.

![set-channel-status](/assets/static/set-channel-players.webp)

## 3. 서버 등록
* 게임서버가 등록되려면 등록되는 시점에는 온라인이어야 해요.

1. [/서버등록](/commands/server-register) 명령어로 게임의 종류, IP를 입력하여 등록해요.

![add-server](/assets/static/add-server.webp)

2. 포트 옵션은 기본 포트가 지정되나, 혹시 다르게 세팅하셨다면 지정하여 주세요.

![add-server-withport](/assets/static/add-server-withport.webp)

3. 정상적으로 등록되었다면 이런 메세지가 표시될 거에요.

![add-server-complete](/assets/static/add-server-complete.webp)
