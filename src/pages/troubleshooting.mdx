# 문제 해결
## 에러 코드
| 코드 | 메세지 | 비고 |
| - | - | - |
| 400 | 잘못된 요청 | 명령의 인자로 잘못된 요청을 받았어요. |
| 401 | 권한 없음 | 해당 명령을 수행할 권한이 없어요. |
| 500 | 서버 오류 | 알 수 없는 서버의 오류에요, 계속 발생할 경우 문의해주세요. |
| 800 | 대상 없음 | 명령을 수행할 대상이 없어요. |
| 801 | 데이터베이스 쿼리 실패 | 서버 데이터베이스의 오류에요, 계속 발생할 경우 문의해주세요. |
| 802 | 잘못된 호스트 | 잘못된 호스트 입력 |
| 803 | 잘못된 포트 범위 | 잘못된 포트 입력 |
| 804 | 필수 설정 완료되지 않음 | 채널 설정이 되지 않은 상태에서 명령을 실행하는 경우 발생해요. |
| 805 | 최대 서버 숫자 도달 | 최대 등록 가능 서버는 10개에요. |
| 806 | 재설정 필요 | 채널 등록 절차 재실행이 필요해요. |
| 807 | 디스코드 요청 실패 | 디스코드 서버 오류, 잠시 후 시도해보세요. |
| 1000 | 음성 채널이 아닙니다 | 음성 채널이 아닌 다른 잘못된 채널을 입력했어요. |
